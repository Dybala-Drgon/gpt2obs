# 图标说明

此目录需要放置以下尺寸的插件图标：

- icon16.png - 16x16像素
- icon48.png - 48x48像素
- icon128.png - 128x128像素

## 建议图标设计

可以使用在线工具生成图标：

1. **Favicon Generator**: https://favicon.io/
2. **IconKitchen**: https://icon.kitchen/
3. **Canva**: https://www.canva.com/

推荐使用以下元素：
- 💬 对话气泡
- 📝 文档/笔记
- 💾 保存图标
- 紫色/绿色主题（配合ChatGPT风格）

## 临时方案

在开发测试阶段，可以暂时注释掉 manifest.json 中的 icons 配置，或者使用任意PNG图片重命名为对应尺寸。
